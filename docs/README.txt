These documents are a copy of the official ASTRIAL documentation for schematic and user manual.

For additional documentation on NXP and Hailo see the DOWNLOADS section of www.astrial.ai: 
each release will deliver specific .zip files with all the documents from the vendors.

ASTRIAL_pcb: schematics for the Astrial CM4 SoM.

ASTRIAL_UserManual: details on usage and boot modes.

ASTRIAL_Gpio_Mapping: pin mapping (40 pin GPIO connector) for raspberry-like carriers with CM4.
