example on how to use external MCU connected to main system.

in this case we use ATTiny (with ARDUINO) connected via I2C.

Libraries to be installed for Arduino to compile
 * WS2812B Leds -> Adafruit Neopixel: https://github.com/adafruit/Adafruit_NeoPixel
 * I2C Slave -> TinyWiresS: https://github.com/nadavmatalon/TinyWireS
